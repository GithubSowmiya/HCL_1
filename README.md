# Day Finder in C

This program finds the day of the week for any given date 
using Zeller's Congruence formula.

## How It Works
The program takes day, month, and year as input 
and calculates the correct weekday.

## Sample Input
Enter day: 4  
Enter month: 3  
Enter year: 2026  

## Sample Output
The day is: Wednesday

## Language Used
C Programming
